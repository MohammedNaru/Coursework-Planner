What is a README file?

A README file explains a project’s purpose, setup instructions, and how to contribute. It helps others understand your project quickly.

What is a wireframe?

A wireframe is a simple diagram showing the layout and structure of a webpage or app. It helps plan placement of elements before coding.

What is a Git branch?

A branch in Git allows you to work on a separate copy of the code without affecting the main project. You can develop features or fix bugs safely.
